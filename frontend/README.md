# Frontend (Next.js)

## Scope de este módulo

- Configuración inicial de Next.js con TypeScript.
- Consumo de la API backend para mostrar productos.

## Requisitos

- Node.js (recomendado: LTS)
- pnpm

## Ejecutar

1. Instalar dependencias:

```bash
pnpm install
```

2. Levantar el servidor:

```bash
pnpm run dev
```
